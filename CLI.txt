***** <B> Command Line Interface </B> ******

Basic commands:

Navigate the directories:                       cd  ___dir/folder_Name

Make new folder:                                mkdir Folder_Name

Delete a folder (only empty folders):           rmdir Folder_Name

Create new file:                                touch File_Name.extension

Delete files:                                   rm File_Name1.extension  File_Name2.extension  File_Name2.extension

Details with parameters:                        man + command_name

Move a folder/file                              mv  ___dir/Src_Folder   ____dir/Destination

Create new shortcut/chains of cmd:              alias shortcut='command1;command2;...'    *Note no space between shortcut=expression

List files and folder in directory:             ls [some simple argument: -l details listed -s short]

More advance:

Change file permission                          chmod [permission numbers: using 0,1,2,4 in 3 places] Explained in details
